### **Quick Setup:**

1. **Install Extension**:
   - Press `Ctrl+Shift+X`
   - Search: **"MongoDB for VS Code"**
   - Install (by MongoDB Inc.)

2. **Connect**:
   - Press `Ctrl+Shift+P`
   - Type: **"MongoDB: Connect"**
   - Select: **"Connect with Connection String"**
   - Paste:
   ```
   mongodb+srv://admin:project123@cluster0.7xwslgo.mongodb.net/smart-event-management?retryWrites=true&w=majority
   ```

3. **Explore Your Data**:
   - Click MongoDB icon in sidebar
   - Browse collections: `users`, `events`, `vendors`, `arlayouts`, etc.
   - Run queries, create playgrounds, and visualize your data!
